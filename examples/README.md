# `doppio examples`

## redis-server

Runs on port 6380

```
go run examples/redis-server/main.go
```

- `GET`, `SET`, `DEL`, `QUIT`, `PING`, `SHUTDOWN` commands.  
- Compatible with the [redis-cli](https://redis.io/topics/rediscli) and all [redis clients](https://redis.io/clients).
